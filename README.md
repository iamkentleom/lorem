# Fake GraphQL API

GraphPlaceholder is a fake GraphQL API for testing and prototyping similar to that of JSONPlaceholder

## Usage
Install dependencies
```bash
npm install
```
Run server
```bash
npm run dev # running development server
# or
npm run start # running server
```
